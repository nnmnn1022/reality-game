# DISCORD_BOT_RESPONSIBILITIES

## Bot is Game Master

Discord Bot은 플레이 진행을 담당한다.  
하지만 Bot은 Story를 해석하지 않는다.

## Responsibilities

- Slash Command 처리
- Experience 생성
- 참가자 등록
- Flow 선택 UI 제공
- Scene 전송
- Button/Modal 처리
- Thread 생성
- DM 전송
- Input 수집
- Event 발행
- Renderer Output을 Discord에 반영

## Not Responsible For

- 다음 Stage 결정
- Coverage 판단
- Narrative 생성
- Story Memory 선정
- AI Prompt 작성
