# AI_USAGE_POLICY

## Principle

AI는 마지막 선택이다.

AI는 다음 경우에 사용한다.

- 주관식 Input 해석
- 사진/음성 등 비정형 Input 요약
- Scene Rendering의 표현 개선
- Callback 문장 생성
- Ending Narrative 생성

AI는 다음을 직접 결정하지 않는다.

- 공식 State
- Rule 결과
- Majority
- Timeout
- Visibility
- Participation
- DB 상태 변경

## During Play

AI는 Scene Renderer로 사용될 수 있다.  
말을 많이 하지 않는다.  
현실 플레이를 방해하지 않는다.

## After Play

AI는 Narrative Director로 사용될 수 있다.  
Story Memory를 연결해 Ending을 생성한다.
